# Math teacher

## SUMMARY

Math teacher is a software to train your math skills. Currently only multiplication is implemented. The software is maily developed for SailfishOS but an Android version is also available.

The whole software is licensed under BSD.

## PRIVACY
As a maintainer of this code I do not collect any information of you. All the data is locally on your phone.

The app is not sending any information of your identity to me. I might get some general information of the app users depending on the store you installed the app.

I do not install any adds to this app.

The source is always downloaded to Github before the app excecutable is sent to the store.

## KNOWN ISSUES (ANDROID)

Not all the features are functioning on Android phones.
